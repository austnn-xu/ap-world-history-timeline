# AP World History Timeline

A minimal interactive AP World History: Modern timeline app.

## Open Locally

Open `index.html` in a browser. Keep these files and folders together:

- `index.html`
- `styles.css`
- `app.js`
- `assets/`

## Features

- Vertical AP World timeline from Unit 1 to Unit 9
- Dark mode
- Search that guesses the best AP unit for a term
- Click-to-lock topic cards
- Per-topic image cards
- Study panel with key points and vocabulary

## Image Sources

Images were downloaded locally for this educational project from Wikimedia Commons and Wikipedia page thumbnails. Credits collected by the downloader scripts are in:

- `assets/apw/credits.json`
- `assets/wiki/credits.json`
